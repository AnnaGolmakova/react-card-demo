## Technologies Used

- React
- Typescript
- HTML5
- Tailwind
- Vite

## Running locally and development

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
3. Run the development server:
   ```
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser
